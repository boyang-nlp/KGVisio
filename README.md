# KGVisio: A browser-based knowledge graph visualization system
## Requirements
   - Python enviroment: 
		python >= 3.6
		py2neo >= 2.0
		BeautifulSoup >= 4.6.0
		tqdm >= 4.21.0
		flask >= 0.12.2
   - Graph database
     neo4j(graph database) >= 3.1.2
   - OS
     windows 10
## Basic useage
- Import data
   ``` python import2db.py```
- Run server
  ``` python server.py```
